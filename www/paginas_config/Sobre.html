Aplicativo CargoTracer - Solução Inovadora para Roteirização de Entregas para Caminhões

O desenvolvimento do aplicativo CargoTracer surgiu como um projeto de Trabalho de Conclusão de Curso (TCC) no curso de Desenvolvimento de Sistemas, sendo idealizado e executado por três estudantes: Allan José, Fernando de Carvalho e Mateus Martins. Nosso objetivo foi criar uma solução prática e eficiente para o problema que empresas de pequeno e médio sofrem ao tentarem monitorar suas entregas de forma não automatizada.

Funcionalidades Principais

  • Interface Amigável: Desenvolvemos uma interface intuitiva para garantir uma experiência de usuário agradável e acessível.
  • Roteirização: Roteirização especifica para caminhões, coisa que nenhum outro aplicativo faz.
  • Emissão de CT-es: Emissão de CT-es automatizada, agilizando os procedimentos de finalização das entregas do dia.
  • Integração com Serviços Externos: Integração com APIs e serviços para ampliar as capacidades do aplicativo e oferecer mais valor aos usuários.

Tecnologias Utilizadas

As principais tecnologias empregadas incluem:

  • Linguagens de Programação: Javascript.
  • Frameworks e Bibliotecas: Jquery, Apache Cordova e Framework7.
  • Banco de Dados: MySQL.

Desafios e Soluções

Durante o desenvolvimento, enfrentamos desafios como Incompatibilidade de Api's e busca por metodos compativeis, que superamos através de Estudo e Disciplina.

Conclusão e Perspectivas Futuras

O aplicativo CargoTracer não só cumpriu os requisitos do nosso TCC, como também abriu portas para futuras melhorias e expansões. Planejamos continuar o desenvolvimento com base no feedback dos usuários e explorar novas funcionalidades que possam agregar ainda mais valor.

Este projeto representou uma oportunidade de aplicar na prática os conhecimentos adquiridos durante o curso, reforçando nossas habilidades técnicas e colaborativas, e nos preparando para os desafios profissionais que estão por vir.
